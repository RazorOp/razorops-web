Razorops is the easiest CI/CD tool to set up

About Nimble

Nimble Buying is a cognitive B2B e-commerce platform with expertise in providing end-to-end sourcing and supply chain solutions of engineering products to global manufacturing companies.

The Challenge

Nimble Buying development team wanted the power to work independently and to push code to production without any delays.They also wanted to avoid the long set up time and mainly the maintenance commonly involved in CI/CD pipeline.

Solution

Nimble Buying development team had a solution Razorops at hand that integrates with GitHub/Gitlab. Now they are able to deploy code directly into production, and it only takes a few minutes to go live. Now Every update they make is covered by the Razorops CI/CD pipeline. The build process which used to take more than thirty minutes now takes less than half that time. Using Razorops in their development process made testing more of an essential part of their workflow now.
