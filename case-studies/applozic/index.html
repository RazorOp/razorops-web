Moving fast, making customers happy with Razorops

Why Razorops ?

Razorops is a Saas based CI/CD platform which helps to be 100% operational on a technical level so we can focus on delivering the best product in a short amount of time.

About Applozic

Applozic is a Real-Time Chat and Messaging API for Developers which makes communication easy from both the web or mobile with push notification, video and voice, and cross-platform integration.

The Challenge

The Developers were on weekly releases, so they needed fast iterations. They wanted to avoid the maintenance and setup time for deployments which commonly involved continuous integration as well as the learning curve needed for new engineers joining.

They needed a DevOps solution for :

  • Ensure faster deployments.
  • Stabilize the work environments
  • Significant improvement in product quality
  • Continuous delivery of software

“Razorops is one of my stress free tool and we were able to increase concurrency, deliver value faster, and our feedback loop was faster.”
- Devashish Mamgain ( Founder of Applozic )

Solution

The ultimate goal was to have an automated testing and deployment pipeline that would allow the team to to focus on development rather than worrying about deploying the bits. By shifting over to Razorops, Applozic has seen a significant increase in its ability to auto-balance its test run times and deployment - continuously rebalancing which tests are running on which nodes to give Applozic the shortest possible time from merge to deploy.
