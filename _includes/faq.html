Frequently Asked Questions

Q: How do I get started?

Getting started with Razorops is easy, you can have a look quick look on Documents and signup for razorops.

1. Signup Razorops, Create your first Pipeline project
2. Add .razorops.yaml to your source code root directory
3. Define your Pipeline (Click here to Learn)
4. Commit and push your source code
5. This new commit will trigger your pipeline
6. Still need help Schedule a Demo

Q. Which source code management services are supported?

Razorops supports major code-hosting services like GitHub, Bitbucket and Gitlab

You can find Razorops at Github marketplace as well.

Q. How do you handle upgrading or canceling a subscription?

You can change or cancel your subscription at any time from your dashboard.
