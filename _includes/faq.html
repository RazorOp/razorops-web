Frequently Asked Questions

Q: How do I get started?

Answer

Getting Stared with Razorops is easy, you can have a look quick look on Documents and signup for razorops.

  1. Signup Razorops
  2. Create Pipeline
  3. Add .razorops.yaml to your source code root dir
  4. Define your Pipeline (Click here to Learn Pipeline)
  5. Commit and push your source code
  6. This new commit will trigger your pipeline

Q: Which source code management services are supported?

Answer

Razorops supports major code-hosting services like GitHub, Bitbucket and Gitlab

You can find Razorops at Github marketplace as well.
